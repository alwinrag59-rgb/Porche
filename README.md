# Porsche Showcase

A high-end scrollytelling website for the Porsche 911 GT3 RS.

## Features
- Interactive 3D/Scrollytelling elements
- High-end automotive aesthetic
- Next.js App Router
- React, Tailwind CSS, Framer Motion

## Getting Started

First, navigate to the app directory:
```bash
cd porche-app
```

Then, run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
